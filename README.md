# Organização e limpeza de dados do DataSUS Tabnet
Este é um código simples para ajudar na organização e limpeza dos dados baixados do **DataSUS Tabnet**. Aqui, utilizamos dados de casos de um agravo por município por mês para 5 anos. O *script* foi feito para a disciplina de Impactos das Mudanças Climáticas na Saúde do PPGICS/ICICT/Fiocruz.
Link para o DataSUS Tabnet: https://datasus.saude.gov.br/informacoes-de-saude-tabnet/

English version is under development.
# DataSUS Tabnet Data Cleaning and Organisation
This is a simple piece of code to assist with the cleaning and organisation of data downloaded from DataSUS Tabnet. Here, we use case data for a specific health condition per municipality per month across 5 years. This script was designed for the course Impacts of Climate Change on Health at PPGICS/ICICT/Fiocruz.
