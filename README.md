# Organização e limpeza de dados do DataSUS Tabnet
Este é um código simples para ajudar na organização e limpeza dos dados baixados do **DataSUS Tabnet**. Aqui, utilizamos dados de casos de um agravo por município por mês para 5 anos. O *script* foi feito para a disciplina de Impactos das Mudanças Climáticas na Saúde do PPGICS/ICICT/Fiocruz.
Link para o DataSUS Tabnet: (https://datasus.saude.gov.br/informacoes-de-saude-tabnet/) [https://datasus.saude.gov.br/informacoes-de-saude-tabnet/]
